# QPG6200 IoT Software Development Kit

Welcome to the official Qorvo® IoT Software Development Kit (SDK) for the QPG6200 System-on-Chip.

Learn about [QPG6200 Features](https://qorvo_sdk.gitlab.io/public/devkits/qpg6200-iot-sdk/sphinx/docs/QORVO.html) and [see available products on Qorvo.com](https://www.qorvo.com/search?mode=2&key=&des=QPG6200&exact=&any=&none=&pageNumber=1&pageSize=10&tbl=ta0132).


To get started, [follow the Quickstart tutorial](https://qorvo_sdk.gitlab.io/public/devkits/qpg6200-iot-sdk/sphinx/docs/QUICKSTART.html).

QPG6200 provides some unique features that you may want to explore:
- **ConcurrentConnect** - a no-delay multiprotocol connectivity (Matter, Zigbee, Thread, Bluetooth® Low Energy).
   - [Test it with Concurrent Light sample app](./Applications/Concurrent/Light/) or [read the documentation](https://qorvo_sdk.gitlab.io/public/devkits/qpg6200-iot-sdk/Applications/Concurrent/Light/README.html).
- **Antenna Diversity** - dual antennas for increased range and reliability.
   - [Test it with Radio Control Console](./Tools/RadioControlConsole/) or [read the documentation](https://qorvo_sdk.gitlab.io/public/devkits/qpg6200-iot-sdk/Tools/RadioControlConsole/README.html).
- **Competitive power efficiency**.
   - [Test it with Battery Monitor sample app](./Applications/Peripherals/Batterymonitor/) or [read the documentation](https://qorvo_sdk.gitlab.io/public/devkits/qpg6200-iot-sdk/Peripherals/Batterymonitor/README.html).

For more RF testing options, refer to:
- [Getting Started with Radio Control Console](https://qorvo_sdk.gitlab.io/public/devkits/qpg6200-iot-sdk/Tools/RadioControlConsole/README.html)
- [Product Test Component](https://qorvo_sdk.gitlab.io/public/devkits/qpg6200-iot-sdk/Applications/PTC/README.html)
