qvIO
====

.. _qvio_warning:

.. warning::
    The qvIO module is soon to be replaced with the dedicated software modules, eg. :ref:`shared_button_handler`, and :ref:`shared_status_led` modules.


The qvIO is an all in one module that allows you to:

* React to button presses and releases.
* Control various type of LEDs:
    * PWM controlled LEDs
    * GPIO controlled LEDs
    * qDrvLED controlled LEDs
* Read and send data over UART.
* Read from temperature sensor embedded in the Qorvo\ |reg| SoC.
